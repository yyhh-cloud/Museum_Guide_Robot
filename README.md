# 🏛️ 博物馆智能导览机器人 (Museum Guide Robot)
基于 STM32 和 K210 的博物馆智能导览机器人，支持 AprilTag 视觉导航。


## ✨ 功能亮点
* **👀 视觉导航**：利用 K210 识别 AprilTag (二维码)，根据 ID 判断当前位置。
* **🧠 智能决策**：STM32 内置状态机，支持按键切换目标展品（蒙娜丽莎 / 星空）。
* **🔄 自动变道**：识别到路口路标 (Tag 0) 时，自动欺骗底盘 PID 进行转向。
* **📏 精确停车**：基于视觉焦距公式计算距离，误差控制在厘米级。

## 🛠️ 硬件清单
* **主控**：STM32F103RCT6 (C10B 传感器板)
* **视觉**：K210 (MaixBit / CanMV)
* **底盘**：麦克纳姆轮 + C50C 驱动板
* **传感器**：MPU6050 陀螺仪 + 超声波测距

## 🚀 如何使用
1.  **上电**：OLED 显示 `Mode: Patrol` (巡逻模式)。
2.  **选择目标**：
    * 按一下按键：前往 **蒙娜丽莎** (Tag ID: 1)。
    * 再按一下：前往 **星空** (Tag ID: 2)。
3.  **运行**：小车将自动巡线，并在路口自动选择方向，到达目标前 45cm 自动停车。

## 📂 文件说明
* `K210_Script/main.py`: 运行在 K210 上的 Python 视觉代码。
* `STM32_Project/`: Keil MDK5 工程源码。
    * `main.c`: 核心控制逻辑与状态机。
    * `k210.c`: 视觉数据解码协议。
